Power2Go

Powerbank lending system.

This project focused on creating a powerbank loaning system where users can sign up for an app, scan the qr code on a physical powerbank station and take out a powerbank. They can be returned to any of the powerbank stations.

Features

Quick powerbank lending by using a qr codePricing based on length of usage

Contributions

Handled the programming of the physical powerbank station slots.

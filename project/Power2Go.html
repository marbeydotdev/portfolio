Power2Go

Powerbank lending system.

This project focused on creating a powerbank loaning system where users can sign up for an app, scan the qr code on a physical powerbank station and take out a powerbank. They can be returned to any of the powerbank stations.

The project infomercial, in Dutch.

Features

Lend powerbanks by scanning a QR code on a physical powerbank station.View nearby stations sorted by distance in the app.Fully automatic powerbank dispensing.

Contributions

Handled the programming of the physical powerbank station slots, running on a Raspberry Pi 4b.Designed and implemented the front-end.Created the infomercial.
